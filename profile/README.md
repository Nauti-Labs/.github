# Nauti-Labs

**AI-first software studio building intelligent systems for real-world operations.**

We design, build, and deploy autonomous agents, computer vision pipelines, and full-stack platforms — with a specialty in digitizing industries that still run on paper.

---

### What We Build

**AI & Agents** — Multi-agent orchestration, human-in-the-loop approval workflows, autonomous task execution with Claude API integration

**Computer Vision** — Hybrid client/server pipelines using TensorFlow.js + U2-Net/SAM for real-time detection, segmentation, and visual intelligence

**Full-Stack Platforms** — Next.js, Flask, FastAPI backends powering operational dashboards, form automation, and data processing at scale

**ML From Scratch** — Neural networks built without frameworks — single neuron to transformer architecture in pure Python
